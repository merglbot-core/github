# Review Health Longform Notes

This document provides a longer documentation-only maintenance note. It is
plain text intended to exercise ordinary review handling for a larger markdown
change.

## Review Principles

The repository should keep review notes concrete and tied to current pull
request evidence. Review summaries should identify the current head, the checks
that were inspected, and whether the result is actionable. If evidence is old,
the summary should say so clearly.
